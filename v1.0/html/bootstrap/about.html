A Bit About Us

The Enhanced IP (EnIP) Project is an effort to create novel
techniques to enhance IPv4's addressing capabilities by using
IP options. Ultimately, this effort is about creating a larger
Internet address space with a simple transition plan.

Thus far the Enhanced IP project has:

  • A Research Paper
  • Linux Kernel Code
  • Virtual Machines in Downloads
  • Live network between UMD and UDel
  • Demo of Enhanced IP NAT in a LTE System
  • Demo of Enhanced IP on an Android Phone
